# Change Log

All notable changes to the "claudeos-chat" extension will be documented in this file.

## [0.1.0] - 2026-03-09

### Added
- Initial release of ClaudeOS Chat extension
- Chat panel webview in VS Code sidebar
- WebSocket connection to ClaudeOS kernel server
- Commands for opening chat in editor and creating new sessions
- Configuration settings for WebSocket URL and auto-connect
- Activity bar integration with custom icon

### Planned
- Expandable thought chains UI
- Tool call inspection
- Notification badges for user input required
- Session management integration
- Slash command support
